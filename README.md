# Numerical Analysis

數值分析作業：二維牛頓法

在這個專案中，我使用 C++ 實作了二維牛頓法，並利用 Python 的 Matplotlib 套件進行視覺化展示。專案包含以下功能：

1. **函數圖形繪製**：繪製目標函數的等高線圖，幫助理解函數的形狀和特性。
2. **收斂路徑動畫**：展示牛頓法從初始點到收斂點的過程，直觀地顯示算法的迭代過程。
3. **根的分佈圖**：顯示不同初始點收斂到不同根的區域。
4. **迭代次數圖**：展示不同初始點收斂所需的迭代次數。

## 檔案結構

-   `01257128_hw1_root2.pdf`: 我的作業 A4 報告檔案
-   `images/`: 存放生成的圖片和動畫
-   `include/`: 相關自訂類型的標頭文件
-   `src/`: 包含所有源碼文件
    -   `cpplib/`: C++ 的多項式類別、牛頓法相關實作
    -   `numerical/`: Python 繪圖的實作
    -   `main.py`: 主程式入口，負責調用各模組完成任務
    -   `newton_cli.cpp`: C++ 主程式入口，負責處理命令列參數並調用牛頓法
-   `makefile`: 用於編譯 C++ 程式碼
-   `pyproject.toml`: Python 專案配置文件
-   `README.md`: 專案說明文件

## 安裝與使用

1. 編譯 C++ 程式碼：
    ```bash
    make
    ```
    編譯完成後的執行檔為 `newton_cli`。
2. 安裝相依套件：
    ```bash
    poetry install
    ```
3. 執行程式：
    ```bash
    python src/main.py
    ```
